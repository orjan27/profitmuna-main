## STRIDE Threat Register

| Threat ID | Category               | Component               | Disposition | Mitigation Plan                                                                                                                                           |
| --------- | ---------------------- | ----------------------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| T-6-03    | Elevation of privilege | GET/PUT /api/settings   | mitigate    | `.use('/*', requireAuth)` at router level; every query scoped `eq(users.id, c.get('userId'))` — no settings read/write without auth, no cross-user access |
| T-6-04    | Tampering              | settings update body    | mitigate    | zValidator('json', updateSettingsSchema) at route entry; enums + int bounds reject invalid currency/frequency/day/hour before service call                |
| T-6-05    | Information disclosure | settings-service errors | accept      | Service throws HTTPException 404 only (no row); generic shape via app.onError — no internal details leaked                                                |
| T-6-SC    | Tampering              | npm installs            | accept      | No new dependencies (all shadcn components already installed per UI-SPEC §Registry Safety)                                                                |
