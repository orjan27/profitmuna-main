| --------- | ---------------------- | -------------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |

| T-05-02   | Tampering              | from/to date params                    | mitigate    | Zod regex /^\d{4}-\d{2}-\d{2}$/ on from/to + z.coerce.number().int().min/max on feedPage/feedSize at route entry (422 on invalid)                       |
| T-05-03   | Elevation of Privilege | All aggregate + feed queries           | mitigate    | Every query gated by eq(table.userId, userId) where userId = c.get('userId') from the JWT; cross-user isolation asserted in dashboard.test.ts (no IDOR) |
| T-05-04   | Information Disclosure | Wallet balance + feed cross-user joins | mitigate    | walletTransactions and wallet-balance queries add eq(...userId, userId) before any join; feed wallet rows join only the user's own wallets              |
| T-05-05   | Information Disclosure | Error responses / logs                 | accept      | Service throws HTTPException with code-only messages; unexpected errors propagate to app.onError (generic 500). No row data or PII logged               |
