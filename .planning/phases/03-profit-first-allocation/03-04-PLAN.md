<threat_model>

## Trust Boundaries

| Boundary                                 | Description                                       |
| ---------------------------------------- | ------------------------------------------------- |
| editor/form inputs → server action → API | User-entered name, percent, color                 |
| client UI delete-disable                 | UI affordance only — not an authorization control |

## STRIDE Threat Register

| Threat ID | Category               | Component                  | Disposition | Mitigation Plan                                                                                                                                                       |
| --------- | ---------------------- | -------------------------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| T-03-05   | Tampering              | percent input range        | mitigate    | Number inputs bounded 0-100; server action converts to bp; API Zod (Plan 02) re-validates int 0-10000 — client bounds are convenience, server is authoritative        |
| T-03-06   | Tampering (XSS)        | account name in form/cards | mitigate    | Names rendered as text content; React escapes; API trims/length-caps via Zod                                                                                          |
| T-03-07   | Elevation of Privilege | UI delete-disable bypass   | mitigate    | UI disabling Delete for non-CUSTOM is convenience only; the server `deleteAccount` guard (`accountType !== 'CUSTOM'`, Plan 02) is the real control — defense in depth |
| T-03-11   | Tampering              | arbitrary color value      | mitigate    | Swatch palette restricts choice to PF_DEFAULT_COLORS; API `z.enum(PF_DEFAULT_COLORS)` (Plan 02) rejects anything else                                                 |
| T-03-SC   | Tampering              | npm installs               | accept      | No new packages; no install tasks this plan                                                                                                                           |

</threat_model>
